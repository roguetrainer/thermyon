# Geometric Interpretation of Code Switching: PG(2,2)→PG(3,2) Projective Envelopes, Transversal CCZ, and a Geometric Primitive for Grover's Diffusion Operator

**Paper:** 210 | **Portfolio C** — Quantum Hardware

**DOI:** [10.5281/zenodo.19929360](https://zenodo.org/records/19929360)

## Abstract

Code switching from the Steane code (PG(2,2)) to the Reed-Muller code (PG(3,2)) is given a projective geometry interpretation. The Fano plane envelope provides a geometric primitive for the CCZ gate and Grover diffusion.
