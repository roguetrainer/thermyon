# The ζ(21) Apéry Generalization: Cubic Pisot Units, G₂ Geometry, and a Conjectured 4-Term Recurrence

**Paper:** 265 | **Portfolio E** — Grand Challenges

**DOI:** [10.5281/zenodo.20029647](https://zenodo.org/records/20029647)

## Abstract

Conjectures a 4-term recurrence for ζ(21) analogous to Apéry's recurrence for ζ(3), motivated by cubic Pisot unit geometry and the G₂ weight lattice.
